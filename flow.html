Workflow

私たちは「やりっぱなし」の研修は行いません。
徹底的な分析から、組織の次の一歩に繋がるフィードバックまで責任を持って伴走します。

01

事前ヒアリングと
研修デザイン

まずは、依頼主様（経営層・人事担当者様など）と丁寧な打ち合わせを行います。 組織の現状課題を抽出し、「この研修を通じてどうなりたいか」というゴール地点を明確に設定。 半日から丸2日間までのスケジュールに合わせて、最適なワークショップをオーダーメイドでデザインします。

主な内容： 現状分析 / ゴール設定 / 参加者構成の検討 / 研修プログラムの設計
02

対面・オフライン
ワークショップ

100%対面での研修を行います。私たちは「身体感覚」を通じた対話を重視するため、オンラインでは行いません。 シンギュラリティ高校の教育現場で培われたファシリテーションにより、LEGO®ブロックを通じて全員の潜在意識をテーブルの上に引き出します。 全国どこへでも伺い、集中的な「問い」への探究をサポートします。

主な内容： 100% LEGO® SERIOUS PLAY® メソッド / チームビルディング / 戦略・ビジョンの可視化
03

集約・フィードバック
レポートの提供

研修終了後、参加者の間でどのような意見があり、どのようなビジョンが描かれたのかを丁寧にまとめます。 情報のプロフェッショナルであるファシリテーターが、単なる記録ではない、組織の「加速度的な成長」に繋がるためのフィードバックを行います。

主な内容： 研修結果のサマリー / 潜在課題の抽出 / 次のアクションへの提言
